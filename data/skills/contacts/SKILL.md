# contacts

Store and look up contacts — names, birthdays, phone numbers, addresses, and notes.

## When to use

- "Save John's number as 555-1234"
- "When is Sarah's birthday?"
- "What's Mom's address?"
- Looking up any saved contact information

## Notes

- Stored locally as JSON
- Contacts can have: name, phone, email, birthday, address, relationship, notes
- Search by name or any field
- Complements the memory system — use contacts for people's detailed info, memory for quick facts
