# notes

Quick-capture notes, ideas, and snippets. Stored as searchable markdown files.

## When to use

- User says "note this down", "save this", "jot down"
- Capturing meeting notes, ideas, reading lists, thoughts
- Searching past notes: "what did I note about X?"
- Different from memory (key-value facts) — notes are freeform text

## Notes

- Each note is a markdown file in the notes directory
- Notes are tagged and timestamped for easy retrieval
- Supports full-text search across all notes
- Can append to existing notes or create new ones
