# stock-tracker

Check stock prices, market data, and portfolio performance. Uses Yahoo Finance (no API key required).

## When to use

- "How's AAPL doing?" or "check TSLA price"
- "How's the market today?"
- Portfolio check or watchlist queries
- Any stock/index/ETF price lookup

## Notes

- Supports individual stocks, ETFs, indices, and crypto
- Returns current price, daily change, and key metrics
- Can check multiple symbols at once
