# expense-tracker

Log, categorize, and summarize personal expenses via quick messages.

## When to use

- "Spent $45 on groceries" or "lunch was $15"
- "How much did I spend this month?"
- "Show my spending by category"
- Any expense logging or budget review

## Categories

food, groceries, transport, shopping, entertainment, health, kids, home, utilities, subscription, other

## Notes

- Amounts default to USD
- Each expense: amount, category, description, date
- Supports monthly/weekly/custom range summaries
- Can auto-categorize based on description keywords
