# translator

Translate between languages. Handle translation directly — no external API needed.

## When to use

- User sends text in one language and wants another
- "How do you say X in [language]?"
- Translating messages, documents, emails
- Explaining nuances, idioms, or cultural context

## Guidelines

- Preserve tone and register (formal vs casual)
- For technical terms, provide both the translation and the original term in parentheses
- For idioms, explain the meaning rather than translating literally
- Adapt language complexity to the audience (simpler for children, etc.)
