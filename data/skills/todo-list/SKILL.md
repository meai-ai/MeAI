# todo-list

Personal task management. Add, complete, list, and organize tasks with optional priorities and due dates.

## When to use

- User says "add to my todo" or "I need to..."
- Listing pending tasks: "what do I need to do?"
- Completing tasks: "done with X" or "finished X"
- Any personal action item tracking

## Notes

- Tasks stored locally as JSON, organized by project/category
- Supports priorities: high, medium, low
- Supports optional due dates
- Tasks can be filtered by status, priority, or category
