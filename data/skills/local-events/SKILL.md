# local-events

Discover local events and activities in the user's area. Supports kid-friendly filtering.

## When to use

- "What's happening this weekend?"
- "Any kid-friendly events near us?"
- "Find things to do"
- "Any tech meetups this week?"
- Seasonal activities: farmers markets, festivals, holiday events

## Sources

- Local event listings and calendars
- Eventbrite
- Meetup.com

## Notes

- Default location: based on user's city from character config
- Can filter by: family/kids, tech, outdoors, arts, food
- Results include title, date, location, link, and short description
