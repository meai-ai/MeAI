# news-digest

Fetch and summarize news from RSS feeds. Covers tech, general news, and finance.

## When to use

- User asks "what's the news?" or "any tech news today?"
- Morning briefing requests
- Checking specific topics or feeds

## Available feeds

- **tech**: Hacker News, TechCrunch
- **general**: BBC World, NPR
- **finance**: Yahoo Finance, MarketWatch

## Notes

- Returns headlines with links, grouped by category
- Ask the user which categories they care about and remember their preference
