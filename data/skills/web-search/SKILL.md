# web-search

Search the web and fetch page content. Uses DuckDuckGo (no API key required).

## When to use

- User asks a factual question you're unsure about
- Anything requiring current/recent information (news, events, prices, docs)
- Looking up documentation, tutorials, or technical references
- Researching local events, restaurants, activities

## Notes

- Returns top search results with titles, URLs, and snippets
- Can also fetch and extract text from a specific URL
- Prefer this over guessing when factual accuracy matters
