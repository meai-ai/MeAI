# calculator

Evaluate math expressions, unit conversions, and currency exchange. More reliable than mental math.

## When to use

- Any arithmetic: "what's 15% tip on $86?"
- Unit conversion: "how many km is 5 miles?"
- Currency: "200 CNY to USD"
- Date math: "how many days until Dec 25?"
- Percentage calculations, splits, compound interest

## Notes

- Uses safe expression evaluation (no eval)
- Supports basic arithmetic, percentages, powers, roots
- Built-in unit conversions (length, weight, temperature, volume)
- Currency rates fetched live from free API
